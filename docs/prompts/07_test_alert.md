# Prompt 07: Submitted Data Alert

Use this prompt with Claude Code.

```text
Add the assignment-required temporary alert for successful submissions.

Requirements:
1. Implement showSubmittedDataAlert(formData).
2. The alert must display:
   - Name
   - Email
   - Phone
   - Message
3. The alert must use the values submitted by the user before the form is cleared.
4. The alert must appear only after all fields pass validation.
5. The alert must not appear for invalid submissions.
6. Do not log the submitted data to the console.
7. Keep a code comment explaining that this alert is only for the assignment and would be replaced by backend submission in a real project.

Expected alert format:

Submitted Data:

Name: Taylor Smith
Email: taylor@example.com
Phone: +972541234567
Message: I would like more information.

After implementation, explain where the alert is called in the submit flow.
```
